hiveplotter

A python library for creating highly customisable hive plots from NetworkX graphs.

This project is released under the BSD License.

WIP

Requires:

  • LaTeX
  • Dependencies as listed in setup.py: shapely, networkx, pyx, numpy, PIL

See hiveplotter/examples for usage.

Default behaviour can be found in hiveplotter_defaults.ini Default behaviour can be overridden with kwargs or by pointing to an alternative config file. Behaviour can be changed between instantiation of the hive plot and drawing.
